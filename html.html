What is HTML?


HTML is the standard markup language for creating Web pages.
  • HTML stands for Hyper Text Markup Language
  • HTML describes the structure of Web pages using markup
  • HTML elements are the building blocks of HTML pages
  • HTML elements are represented by tags
  • HTML tags label pieces of content such as "heading", "paragraph", "table", and so on
  • Browsers do not display the HTML tags, but use them to render the content of the page


Sample Code



          <!DOCTYPE html>
            <html>
              <head>
                <title>Page Title</title>
              </head>
              <body>

                <h1>My First Heading</h1>
                <p>My first paragraph.</p>

              </body>
            </html>
  • The </!DOCTYPE html> declaration defines this document to be HTML5
  • The </html> element is the root element of an HTML page
  • The </head> element contains meta information about the document
  • The </title> element specifies a title for the document
  • The </h1> element defines a large heading
  • The </p> element defines a paragraph

Sample Code



          
<tagname>content goes here...</tagname>
  • HTML tags normally come in pairs like <p> and <p>
  • The first tag in a pair is the start tag, the second tag is the end tag
  • The end tag is written like the start tag, but with a forward slash inserted before the tag name

CSS


Designs with HTML (HTML is just the basic structure, but what if you need to design it?)

Internal
<style type="text/css">
   .myNewStyle {
      font-family: Verdana, Arial, Helvetica, sans-serif;
      font-weight: bold;
      color: #FF0000;
   }
</style>

External
Write that in a separate file.css and then import it

style.css:
.myNewStyle {
   font-family: Verdana, Arial, Helvetica, sans-serif;
   font-weight: bold;
   color: #FF0000;
}

index.html:
<link rel="stylesheet" href="style.css">


Bootstrap


Isn't writing all those tags annoying? Bootstrap has built-in CSS that you can import

  1. Download the files from https://getbootstrap.com/docs/4.0/getting-started/download/ to view the CSS
  2. Use CDN (server) to access the CSS remotely

    3. <link rel="stylesheet" href="https://stackpath.bootstrapcdn.com/bootstrap/4.1.3/css/bootstrap.min.css" integrity="sha384-MCw98/SFnGE8fJT3GXwEOngsV7Zt27NXFoaoApmYm81iuXoPkFOJwJ8ERdknLPMO" crossorigin="anonymous">

  3. Access the CSS through the naming of the tags(classes and ids) - If I name <div class="text-blue">, the text within that tag will be blue.

    4. EX:
    
              <div id="nav">
                <nav class="navbar fixed-top bg-dark justify-content-center">
                  <a class="nav-link text-danger" href="#home">Home</a>
                  <a class="nav-link text-warning" href="#grids">Grids</a>
                  <a class="nav-link text-info" href="#buttons">Buttons</a>
                  <a class="nav-link text-danger" href="#table">Tables</a>
                  <a class="nav-link text-warning" href="#other">Other</a>

                </nav>
                </div>
